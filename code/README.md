# Cloned Repositories

1. **[martorell_grid_world](martorell_grid_world/)**
   - URL: https://github.com/mneuronico/Grid-World-Spatial-Orientation-Task
   - Purpose: Official code for "From Text to Space" (arXiv:2502.16690).
   - Key functionality: Grid-world navigation task with different spatial information representations (SIRs).
   - Notes: Contains probing code for LLaMA-3.1-8B.

2. **[stepgame](stepgame/)**
   - URL: https://github.com/ZhengxiangShi/StepGame
   - Purpose: Official code and dataset generation for StepGame (AAAI 2022).
   - Key functionality: Multi-hop spatial reasoning task.
   - Notes: Includes the TP-MANN model.

3. **[spatiallm_stepgame](spatiallm_stepgame/)**
   - URL: https://github.com/Fangjun-Li/SpatialLM-StepGame
   - Purpose: Rectified StepGame benchmark and hybrid reasoning solution (AAAI 2024).
   - Key functionality: Logic-based spatial reasoning combined with LLMs.
   - Notes: Fixes errors in the original StepGame templates.
